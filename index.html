    In the world of Software develepment , managing changes to code of time is crucial. This is where Git comes in . Git is a version control sytesm, a tools that helps devs track , manage and collabore on changes efficiently.

    But what excatly is a version control system?

    A version control system is a software tool that helps developers keep track of changes in their codebase. It allows you to save different versions of your project, go back to previous versions when need . and collaborae with oher teams members

    There are several version control out there i.e:
